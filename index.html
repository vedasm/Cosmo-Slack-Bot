Meet Cosmo,
your Slack buddy.

Cosmo brings cat facts, random jokes, and NASA's Astronomy Picture of the Day straight into your Slack workspace — all through simple slash commands.

View on GitHub See Commands

Commands

Type any of these in your Slack workspace.

🏓
/csb-ping
Check if the bot is running and see response latency.
🐱
/csb-catfact
Get a random cat fact. Great for #random channels.
😄
/csb-joke
Tells a random two-part joke — setup and punchline.
🌌
/csb-apod
Fetches NASA's Astronomy Picture of the Day with a description.
📖
/csb-help
Lists all available commands in the channel.

How to run it

Four steps and you're good to go.

1
Clone the repo
git clone https://github.com/vedasm/Cosmo-Slack-Bot.git then npm install
2
Create a Slack App
Go to api.slack.com/apps, enable Socket Mode, register the 5 slash commands, and install it to your workspace.
3
Add a .env file
Set SLACK_BOT_TOKEN, SLACK_APP_TOKEN, and NASA_API_KEY (free at api.nasa.gov).
4
Start the bot
Run node index.js. You should see bot is running! in the terminal.
